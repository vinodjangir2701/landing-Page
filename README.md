# landing-Page
Convert a Figma landing page design into a responsive webpage 

<div> 1. Key Sections Covered <div/>
------------------------------------------------------------
-> Header & Navigation: Includes logo and navigation buttons with sticky/fixed behavior on scroll.
-> Hero Section: Large introductory banner with headline, subtext, and primary lead forms.
-> Main Content Sections: <section 1> to <section 10> in main structured blocks for features/services, or informational content with a certificate program section.
-> Perfectly match the original Figma mockups.
-> Footer: Contains essential logo and about subtext, copyright info.


2. Techniques Used to Ensure Responsiveness for all devices ( web, tablet, mobile )
------------------------------------------------------------
-> Proper Flexbox & CSS, adaptable section layouts.
-> Media queries to adjust typography, spacing, and layout for tablet and mobile breakpoints.
-> Optimized images using responsive sizes and proper aspect ratios.


3. Challenges Faced / Features Not Implemented
-----------------------------------------------------------
-> Some advanced interactions (e.g., click events or JS-based components) were not included.
