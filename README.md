# landing-Page
Convert a Figma landing page design into a responsive webpage 
<br>
<br>
<div> 1. Key Sections Covered <div/>
<div>------------------------------------------------------------<div/>
<div> -> Header & Navigation: Includes logo and navigation buttons with sticky/fixed behavior on scroll.<div/>
<div> -> Hero Section: Large introductory banner with headline, subtext, and primary lead forms.<div/>
<div> -> Main Content Sections: section 1 to section 10 in main structured blocks for features/services, or informational content with a certificate program section.<div/>
<div> -> Perfectly match the original Figma mockups.<div/>
<div> -> Footer: Contains essential logo and about subtext, copyright info.<div/>

<br>
<br>
<div>2. Techniques Used to Ensure Responsiveness for all devices ( web, tablet, mobile )<div/>
<div>------------------------------------------------------------<div/>
<div> ->Proper Flexbox & CSS, adaptable section layouts.<div/>
<div> ->Media queries to adjust typography, spacing, and layout for tablet and mobile breakpoints.<div/>
<div> -> Optimized images using responsive sizes and proper aspect ratios.<div/>

<br>
<br>
<div>3. Challenges Faced / Features Not Implemented<div/>
<div>-----------------------------------------------------------<div/>
<div> ->Some advanced interactions (e.g., click events or JS-based components) were not included.<div/>
